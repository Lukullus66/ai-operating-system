# Agentic Design Starter

Reusable project template for AI-assisted website design and development.

## Core stack

- Ship Studio
- Next.js
- TypeScript
- Sanity
- Vercel
- GitHub
- Formspark
- Resend

## Optional modules

- multilingual websites
- Instagram post integration
- automatic social publishing for new blog posts
- Facebook publishing
- Relume React components

## Core operating rule

Claude must always think and act like a senior UX designer before acting as a developer.

Claude must:

1. understand the business and user goal
2. analyze the existing project and supplied references
3. propose information architecture, UX and component strategy
4. explain design decisions
5. present an implementation plan
6. wait for approval
7. implement in milestones
8. review accessibility, responsiveness, SEO and performance

## Reference websites

Reference websites may be used to study:

- information architecture
- hierarchy
- spacing rhythm
- typography
- component patterns
- interaction ideas
- content presentation

Claude must never copy protected text, branding, imagery or a complete design. References are inspiration and evidence, not source code.

## Recommended project initialization

1. Copy this folder into a new repository.
2. Rename and complete `PROJECT.md`.
3. Add approved reference websites to `REFERENCE_SITES.md`.
4. Select optional modules in `MODULES.md`.
5. Start Ship Studio with the prompt in `START_PROMPT.md`.
6. Connect Sanity, Vercel, Formspark and Resend only after architecture approval.
