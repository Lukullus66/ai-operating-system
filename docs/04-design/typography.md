# Typography Rules

## Purpose

Typography carries the product voice and establishes hierarchy.

Good typography often matters more than decoration.

## Principles

- Use a clear type scale.
- Use consistent heading levels.
- Keep line lengths readable.
- Maintain sufficient contrast.
- Use font weights intentionally.
- Avoid too many type styles.

## Hierarchy

Every page should make clear:

- what the page is about
- what matters most
- what the user should do next

## Accessibility

Check:

- font size on mobile
- contrast
- line height
- paragraph spacing
- zoom behavior
- logical heading order

## Editorial style

For editorial websites:

- use generous line height
- keep text blocks calm
- use whitespace instead of decoration
- make captions and metadata consistent
