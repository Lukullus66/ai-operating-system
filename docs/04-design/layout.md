# Layout Rules

## Purpose

Layout should create clarity, rhythm and confidence.

Good layout makes content easier to understand.

## Principles

- Use consistent containers.
- Use consistent section spacing.
- Use a predictable grid.
- Avoid arbitrary one-off spacing.
- Design mobile behavior intentionally.
- Avoid visual clutter.

## Section structure

A typical section should have:

- semantic wrapper
- container
- optional eyebrow
- headline
- body copy
- actions
- media or supporting content

## Responsive design

Do not treat mobile as an afterthought.

Check:

- heading line breaks
- image cropping
- card stacking
- navigation behavior
- form usability
- tap targets

## Editorial layouts

For editorial sites:

- prioritize reading rhythm
- use controlled text widths
- create clear hierarchy
- avoid excessive centered text
- let imagery support the story
